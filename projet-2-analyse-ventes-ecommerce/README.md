# 📊 Projet 2 — Analyse de ventes e‑commerce

## 🎯 Objectif
Analyser les ventes d’un site e‑commerce afin d’identifier :
- les produits les plus vendus,
- les périodes de forte activité,
- les comportements clients,
- les leviers d’optimisation.

## 📁 Contenu
- Dataset e‑commerce (commandes, clients, produits)
- Nettoyage des données (Python ou Power BI)
- KPI : CA, panier moyen, taux de conversion
- Visualisations : courbes, bar charts, cohortes

## 🛠 Technologies
- Excel 

## 📈 Livrables
- Rapport d’analyse
- Recommandations business
