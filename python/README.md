# 🐍 Python — Scripts & Notebooks

- Nettoyage des colonnes
- Transformation (création de colonnes dérivées)
- Analyse exploratoire
- Export pour Power BI

### 2. Scripts d’automatisation
Scripts Python permettant :
- d’automatiser la préparation de fichiers CSV,
- de nettoyer des données brutes,
- de générer des rapports intermédiaires pour Power BI.
